# GenoJoin

## Requirements

    - Python >= 3.10
    - Bcftools
    - Gatk4
    - Bgzip
    - Tabix

## Installation

1. Clone git resipotory

   ```
   git clone https://github.com/cmuteam-medcmu/GenoJoin.git
   cd GenoJoin
   ```

2. python installation through conda

   ```
   conda create -n GenoJoin python==3.10
   conda activate GenoJoin
   ```

3. python requirement modules

   ```
   pip install -r requirements.txt
   ```

4. run test script
   ```
   cd test
   bash script.sh
   ```
